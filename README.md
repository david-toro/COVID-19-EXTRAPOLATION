# COVID-19-EXTRAPOLATION

Estimating data of COVID-19 virus

Actually using:

Polynomial regression
Exponential model

Estimating data for confirmed/recovered cases and deaths for the next days in the short-term.

Time is considered in days. Time zero is considered to be the first date in the data.

Graphics with results are exposed in a jupyter notebook, calculations are applied for different provinces/countries as they appear in the data source.

Two general graphs for "Mainland China" and "Out of Mainland China" are included at the end of each set of graphics.    

## Data Source

Data from csv files in https://github.com/CSSEGISandData/COVID-19

## Warning

This code is totally experimental and just a way to automate calculations that i was doing manually with a graphing calculator.
